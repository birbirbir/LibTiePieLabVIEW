================================================================================
  .NET framework bindings for LibTiePie

  LibTiePie is a cross platform library for controllling TiePie engineering USB
  oscilloscopes and arbitrary waveform generators

  See www.tiepie.com/LibTiePie for information about supported instruments,
  platforms, SDK documentation, etc.

  (c) 2014-2015 TiePie engineering
================================================================================

The .NET framework bindings for LibTiePie provide an easy to use interface to 
control oscilloscopes and generators using LibTiePie from any .NET language.


Requirements:

- LibTiePie .NET framework bindings from www.tiepie.com/LibTiePie/DotNET .
  The TiePie.LibTiePie.dll file must be located in the search path of the 
  compiler.

- libtiepie.dll from www.tiepie.com/LibTiePie/Windows .
  For a 64 bit Windows version, the 64 bit library is required.
  For a 32 bit Windows version, the 32 bit library is required.
  The libtiepie.dll file must be located in search path of the compiler.

- .NET framework 2.0 or newer.

Examples:

Examples are available for:

- C# : www.tiepie.com/LibTiePie/Csharp
- VB.NET : www.tiepie.com/LibTiePie/VBdotNET
